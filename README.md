# AnyScript


## 运行环境：

#### 1. tsc（TypeScript Compiler）
**install:** `npm install typescript -g`
**use:** `tsc filename.ts`

#### 2. ts-node
**install:** `npm install ts-node tslib @types/node  -g`
**use:** `ts-node filename.ts`

#### 3. webpack babel
**install:** `npm install webpack webpack-cli -D`
